## 🚀 Sobre a Disciplina
A disciplina de **APC** é a base da computação na UnB. O objetivo aqui é desenvolver o raciocínio lógico para a resolução de problemas complexos e dominar a sintaxe da linguagem C.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem Principal:** C (Padrão C11/C17)
* **Ambiente de Desenvolvimento:** VS Code / Terminal Linux

## 📂 Organização do Repositório
* `Exercicios/`: Resoluções das listas semanais e desafios.
* `Projetos/`: Trabalhos práticos e o projeto final da matéria.
